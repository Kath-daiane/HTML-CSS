Introdução Requisitos Como Usar Suporte

Requisitos

  • Conexão com a internet estável: Para garantir uma navegação fluida, é recomendado o uso de conexões banda larga ou 4G/5G em dispositivos móveis.
  • Navegador moderno e atualizado: Compatível com as últimas versões do Google Chrome, Mozilla Firefox, Microsoft Edge ou Safari. Recomendamos ativar o JavaScript e permitir cookies para pleno funcionamento.
  • Dispositivo compatível: Pode ser acessado por computadores (Windows, macOS, Linux), tablets e smartphones (Android e iOS).
  • Resolução de tela mínima: Para melhor visualização, recomenda-se resolução mínima de 1024x768 pixels. O sistema é responsivo e se adapta a diferentes tamanhos de tela.
  • Conta de usuário ativa: É necessário estar cadastrado e autenticado no sistema para acessar funcionalidades como reservas, empréstimos e renovação de livros.
  • Acesso administrativo (se aplicável): Para bibliotecários e gestores, são exigidas credenciais específicas com permissões avançadas de gerenciamento.
